<bitballs-navigation {app}/>

Creates the navigation for Bitballs.

Parameters

  1. app {AppViewModel}

    The application viewModel. This component will read and set the session property on the AppViewModel.

To create a <game-details> element pass the Session and a Game id like:

<bitballs-navigation
    {app}="." />
